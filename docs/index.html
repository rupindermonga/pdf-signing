How Verification Works

  1. When a document is signed with SealForge, a SHA-256 hash (digital fingerprint) is computed and recorded in the audit trail page and the signing certificate.
  2. Upload the signed PDF here. This tool computes the hash of the file in your browser (nothing leaves your device).
  3. Compare the computed hash with the one on the audit trail. If they match, the document is authentic and unmodified.
  4. If they don't match, the document has been altered after signing — it should not be trusted.
